# Git Learning Repository

This repository is for practicing Git workflows: branching, merging, and rebasing.

## Quick Start
```
git clone [https://github.com/BharatKolhe27/Git-learning.git](https://github.com/BharatKolhe27/Git-learning.git)
cd Git-learning
git checkout -b feature-branch
git add .
git commit -m "My changes"
git push origin feature-branch
